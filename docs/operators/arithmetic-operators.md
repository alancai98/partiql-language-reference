# [](#header-1) Arithmetic Operators

| Operator     | Description |
|:-------------|:------------------|
| `+`           | Add |
| `-`           | Subtract |
| `*`           | Multiply |
| `/`           | Divide |
| `%`           | Modulo |
| `-<number>`   | Negate Number |
