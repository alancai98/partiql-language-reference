# [](#header-1) Conditional Operators

| Operator     | Description |
|:-------------|:------------------|
| `=`           | equal to |
| `!=`/`<>` | not equal to   |
| `<`           | less than      |
| `>`           | greater than |
| `<=`           | less than or equal to |
| `>=`           | greater than or equal to |
| `LIKE`           | check if value is between two values |
| `BETWEEN`           | matches string with wildcard expression |
